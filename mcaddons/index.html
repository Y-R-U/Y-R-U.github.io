How To Install

  1. Hit Download above and grab the .mcaddon file.
  2. Open the file — Minecraft Bedrock imports both packs automatically (Windows, iOS, Android).
  3. Create or edit a world → Behavior Packs → activate the addon (its resource pack comes with it).
  4. Craft, place, troll.
Heads up: Bedrock addons can't read a player's real skin, so the Skin Mocker family uses a creepy mannequin body that takes the name of whoever places it — stalking, teleport rules and hunt mode all work as designed. Needs Minecraft Bedrock 1.21+.
