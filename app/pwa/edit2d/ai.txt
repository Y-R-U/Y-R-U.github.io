Edit2D - 2D Level Editor - AI Reference File
=============================================

## Overview
App: "Edit2D" - A 2D tile-based level and object editor PWA.
Tech: Pure HTML/JS/CSS, ES modules, Canvas 2D API, no frameworks.
Assets: Kenney CC0 2D asset packs (loaded from /gms/assets/kenney/2d/).
Location: /app/pwa/edit2d/

## Directory Structure
app/pwa/edit2d/
  index.html          - Single HTML file: toolbar, canvas, panels, modals
  manifest.json       - PWA manifest (orientation: "any", theme: #1e1e2e)
  sw.js               - Service worker (edit2d-v1, cache-first, dynamic cache for Kenney assets)
  ai.txt              - This file
  css/
    style.css         - Dark theme styles, responsive 3-col layout, modals, panels
  js/
    app.js            - Entry: boot(), SW register, toolbar/menu wiring, mobile drawer
    canvas.js         - Canvas renderer: grid, tiles, objects, camera transforms, culling
    input.js          - Unified pointer events, pinch-zoom, 2-finger pan, scroll wheel, keyboard
    editor.js         - Tool dispatch: brush paint, erase, flood fill, object place/move/delete, collision
    state.js          - Central state object, layer CRUD, undo/redo (delta patches), serialize/deserialize
    assets.js         - Asset catalog loading, sprite sheet loading, placeholder pack generation, tile lookup
    palette.js        - Tile palette panel: search input, category tabs, tile grid thumbnails
    layers.js         - Layer panel: visibility toggle, reorder, add/remove, active selection
    tools.js          - Tool definitions (name, icon, cursor, shortcut), collision type definitions
    io.js             - Export/import JSON, autosave to localStorage, new map, resize, PNG export
    modal.js          - Custom modal system (alert, confirm, prompt, select) - NO js alerts
    utils.js          - Shared: generateId, debounce, throttle, clamp, lerp
  assets/
    catalog.json      - Registry of available asset packs (placeholder + Kenney pack entries)
  icons/
    icon-192.png      - PWA icon 192x192
    icon-512.png      - PWA icon 512x512

Kenney assets location (shared, separate from editor):
/gms/assets/kenney/2d/{packId}/tilemap.png + tilemap.json

## State Shape (state.js)
EditorState = {
  map: { name: string, width: int, height: int, tileSize: int, backgroundColor: string },
  layers: Layer[],              // ordered back-to-front
  activeLayerId: string,
  camera: { x: float, y: float, zoom: float },  // zoom range 0.25-8
  tool: string,                 // "brush"|"eraser"|"fill"|"select"|"object"|"collision"|"pan"
  selectedTileId: string|null,  // e.g. "grass_1", "wall_2"
  brushSize: int,
  showGrid: bool,
  activePackId: string,         // "placeholder", "1bit", etc.
  selection: { x, y, width, height }|null,
  collisionType: int,           // 1=Solid, 2=Platform, 3=Trigger
}

Layer (tile/collision): { id, name, type, visible, locked, opacity, data: 2D array [row][col] }
Layer (object): { id, name, type:"object", visible, locked, opacity, objects: Object[] }
Object: { id, tileId, x, y, width, height, rotation, props, _selected }

History: undoStack/redoStack of delta patches (max 100)
Patch types: tilePaint, collisionPaint, objectAdd, objectRemove, objectMove

## Key Design Notes
- Canvas for viewport, DOM for all UI panels (hybrid approach)
- ES modules (import/export), entry via <script type="module" src="js/app.js">
- No alert()/prompt()/confirm() - all custom modals via modal.js
- Undo/redo via delta-patch history (not full state snapshots)
- Tile IDs are strings ("grass_1") not indices - levels are pack-portable
- Object x,y are floating-point pixel coords (free-position, not grid-locked)
- Autosave: debounced 1s to localStorage key "edit2d_autosave"
- Camera centered on map at boot; zoom toward cursor position
- Grid visible by default (toggle with G key)
- Right-click on canvas = eyedropper (pick tile under cursor)
- Middle-click drag = pan
- Kenney asset packs loaded on-demand, cached dynamically by service worker
- Mobile: bottom drawer palette, slide-in layers panel, 2-finger gestures

## JSON Export Format
{
  "edit2d": "1.0",
  "name": "Level Name",
  "width": 32, "height": 24, "tileSize": 16,
  "backgroundColor": "#1a1a2e",
  "assetPack": "placeholder",
  "layers": [ { id, name, type, visible, opacity, data|objects } ],
  "metadata": { "createdAt": ISO, "editor": "Edit2D v1.0" }
}

Tile data: 2D array [row][col] of string tileId or null
Collision data: 2D array [row][col] of int (0-3)
Object: { id, tileId, x, y, width, height, rotation, props }
