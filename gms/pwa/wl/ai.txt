WARLORDS — AI Project Map
=========================
Turn-based fantasy strategy game (HTML5/JS PWA) inspired by Steve Fawkner's Warlords (1989).
Path: gms/pwa/wl/
Entry point: index.html → js/main.js (ES modules, type="module")

TECH STACK
----------
- Vanilla ES modules (no bundler, no framework)
- Canvas 2D for all game rendering
- DOM overlay for HUD, panels, menus
- PWA: manifest.json + sw.js for offline support

ARCHITECTURE OVERVIEW
---------------------
main.js orchestrates everything: creates World, Camera, Renderer, InputHandler,
GameState, TurnManager, HUD, and AIPlayer instances. Runs a requestAnimationFrame
loop calling renderer.render() each frame.

Game state is stored in World (the source of truth). TurnManager mutates World.
Renderer reads World + Camera + GameState.uiState() to draw each frame.
InputHandler fires onClick/onKeyDown/onWheel callbacks to main.js, which
routes them to TurnManager or GameState updates.

FUNCTION STUBS WITH PARAMS
---------------------------

js/main.js
  init()                                     — boot; attach menu buttons; fake load anim; show menu
  startGame()                                — parse setup form; create World/Camera/Renderer/…; start loop
  buildFactionPicker()                       — render faction cards into #faction-select
  fakeLoad() → Promise                       — animated loader bar; resolves when "done"
  openCityPanel(city, player)                — open city management panel with hire/train callbacks
  showScreen(id)                             — hide all screens; show named screen
  gameLoop()                                 — rAF loop: state.tick(); renderer.render(); renderHUD()
  renderHUD()                                — call hud.updateTurnInfo / updateLog / updateFlash
  onTileClick(tx, ty, button)               — route click to select/move/attack/city/info
  onKeyDown(key)                             — Escape=deselect; Enter/Space=end turn
  onEndTurn()                               — deselect; call turnManager.endTurn()
  onTurnChanged(player)                      — update HUD; trigger AI or enable human controls
  runAI(player)                              — find AIPlayer for player; call ai.takeTurn()
  onCombat(result, atkName, defName)        — show combat overlay; set phase COMBAT_ANIM
  onCombatOk()                              — hide overlay; resume PLAYING phase
  onGameOver(winner)                         — stop loop; show game-over screen

js/data/units.js
  UNIT_TYPES                                 — { [id]: { id, name, attack, defense, move, cost, strength, flying, siege, desc } }
  CITY_PRODUCTION                            — { [cityType]: { gold, units[], maxGarrison, label } }

js/data/factions.js
  FACTIONS                                   — Array<{ id, name, shortName, color, dark, hero, symbol }>
  HERO_NAMES                                 — string[] pool for random hero names

js/game/tile.js
  TERRAIN                                    — { OCEAN|PLAINS|FOREST|HILLS|MOUNTAINS|DESERT|SWAMP: { id, name, color, dark, move, passable, label } }
  TERRAIN_BY_ID                              — TERRAIN values as array (index = id)
  class Tile(x, y, terrain)                 — { x, y, terrain, city, armies[], explored }

js/game/player.js
  class Player(id, faction, isHuman)
    .collectIncome()                          — add city incomes to gold
    .resetArmies()                            — restore movePoints on all armies
    .isAlive() → bool                         — not eliminated and has cities
    .color → string                           — faction.color
    .name  → string                           — faction.name

js/game/city.js
  class City(x, y, type, name, owner)
    .incomePerTurn() → number                 — 0 if unowned; else CITY_PRODUCTION[type].gold
    .availableUnits() → string[]              — unit type IDs trainable here
    .maxGarrison() → number
    .typeLabel() → string                     — 'Village'|'Town'|'City'|'Capital'|'Ruins'|'Temple'
    .exploreRuins() → loot|null               — one-time ruins exploration; returns { type, amount?, unit?, label }
    .isNeutral() → bool

js/game/hero.js
  class Hero(player, name, x, y)
    .gainXP(amount) → bool                   — true if levelled up
    .levelUp()                                — increments level; boosts attack/defense/maxStackBonus
    .maxCommand() → number                    — 8 + maxStackBonus
    .toString() → string
    static random(player, x, y) → Hero       — creates hero with random name
    static hireCost() → number               — 40 gold

js/game/army.js
  class Army(player, x, y)
    .maxMovePoints() → number                — min(unit moves, hero.move)
    .resetMovePoints()
    .hasFlying() → bool
    .hasSiege() → bool
    .totalAttack() → number                  — sum unit attacks + hero bonus
    .totalDefense() → number                 — sum unit defenses + hero bonus
    .maxUnits() → number                     — 8 + hero.maxStackBonus
    .hasRoom() → bool
    .isEmpty() → bool
    .getReachableTiles(world) → Map<"x,y",cost>   — BFS within movePoints
    .findPath(world, toX, toY) → { path, cost }|null  — A* pathfinding
    .mergeFrom(other)                         — absorb other army's units/hero
    .label() → string

js/game/combat.js
  resolveCombat(attacker, defender, city|null) → { attackerWins, remainingAttackerUnits, remainingDefenderUnits, log, rounds, xpGained, cityDefBonus }
    — round-by-round: roll d6 + attack vs d6 + defense+cityBonus; kill weakest loser
  applyCombatResult(attacker, defender, result)
    — writes remaining units back; kills heroes; awards XP

js/game/world.js
  class World(mapSize, numPlayers)
    .generate()                               — calls _buildTerrain → _createPlayers → _placeCities → _giveStartingArmies
    .getTile(x, y) → Tile|null
    .getCityAt(x, y) → City|null
    .getArmiesAt(x, y) → Army[]
    .getFriendlyArmiesAt(x, y, player) → Army[]
    .getEnemyArmiesAt(x, y, player) → Army[]
    .moveArmy(army, toX, toY)                — removes from old tile.armies, adds to new
    .removeArmy(army)                         — removes from tile, world.armies, player.armies
    .captureCity(city, player)               — transfers ownership; updates city/player.cities lists
    .eliminatePlayer(player)                  — sets eliminated; neutralises cities; removes armies
    .checkWinCondition() → bool              — eliminates 0-city players; sets gameOver/winner

js/ai/ai.js
  class AIPlayer(player, world, turnManager)
    .takeTurn()                               — _buyUnits → _moveArmies → optionally _tryHireHero → endTurn

    ._buyUnits()                              — for each city: buy most expensive affordable unit (50% gold budget)
    ._tryHireHero()                           — hire from capital if no hero exists

    ._findNearestEnemyArmy(army) → Army|null

GAME CONSTANTS
--------------
Map: 40×40 (standard) or 56×56 tiles; TILE_SIZE = 32px
Factions: 8 available; 1 human + 2–7 AI
Income per turn: village=2, town=4, city=8, capital=12 gold
Starting gold: 60 per player
Hero hire cost: 40 gold
Combat: round-by-round; each round d6+attack vs d6+defense; loser kills weakest enemy unit
City defense bonuses: village+1, town+2, city+3, capital+4; halved (-2) if attacker has siege unit
Army stack limit: 8 units (+ hero); hero adds +1 every 3 levels
Hero XP per combat: 5 + (rounds/2), capped at 20; XP threshold scales ×1.5 per level


COMMON TASKS
------------
Add a new unit type:       add entry to UNIT_TYPES in js/data/units.js; add to relevant city lists in CITY_PRODUCTION
Add a new terrain type:    add entry to TERRAIN in js/game/tile.js; add draw method to renderer.js
Change AI strategy:        edit _moveArmy priority logic in js/ai/ai.js
Adjust combat balance:     edit resolveCombat() in js/game/combat.js (dice sides, bonuses)
Add new screen:            add hidden div to index.html; add show/hide to showScreen(); style in style.css
Debug world state:         world.armies, world.cities, world.players, world.messageLog
Debug rendering:           renderer._frame, renderer._pulse; uiState passed to render()
