CORSAIR'S FATE - PIRATE 2D ROGUELITE
=====================================
Tech: Vanilla JS (ES modules), HTML5 Canvas, PWA, no build tools
Service Worker: corsairs-fate-v6

FILE STRUCTURE:
  js/game.js      - Main game loop, asset loading, boss spawning, run management
  js/enemies.js   - ENEMY_TYPES[9], Enemy class (healing, mines), EnemySpawner
  js/player.js    - Player class, stats, extraCannons, bossesDefeatedThisRun
  js/combat.js    - Projectiles, mines[], explosive shots, multi-cannon firing
  js/kraken.js    - Kraken boss (procedural drawing, red eye mode, healing)
  js/upgrades.js  - RUN_UPGRADES, PERMANENT_UPGRADES, EXTRA_CANNON_COSTS
  js/ui.js        - HUD, panels, title screen, upgrade panel with extra cannon
  js/story.js     - 3-run story system, RUN_CHAPTERS, boss triggers
  js/particles.js - Particle effects (fire, explosions, text, wreckage)
  js/audio.js     - Procedural SFX + music discovery (parallelized), creepy tracks
  js/debug.js     - Debug panel (GOD, B1, B2, KRK, B? buttons)
  js/input.js     - Touch/mouse/keyboard input manager
  js/world.js     - Procedural world, chunked tiles, ports
  js/trading.js   - Port trading system, buy/sell goods
  js/utils.js     - Math helpers, AssetLoader, formatGold
  css/style.css   - All styling
  sw.js           - Service worker v6, network-first nav, cache-first assets

=== SHIP SPRITE MAP (assets/ships/) ===
24 PNG files in a conceptual 4-row grid:
  Row 1 (1-6):   Undamaged - white, black, red, green, blue, yellow sails
  Row 2 (7-12):  Damage 1  - same 6 colors
  Row 3 (13-18): Damage 2  - same 6 colors (most damaged)
  Row 4 (19-24): Dark/boss ships

Asset key mapping (game.js):
  Player: player_ship=1, player_ship_dmg=7, player_ship_heavy_dmg=13
  Sloop (red): enemy_ship_1=3, damaged_ship_1=9, heavy_dmg_ship_1=15
  Brigantine (blue): enemy_ship_2=5, damaged_ship_2=11, heavy_dmg_ship_2=17
  Galleon (green): enemy_ship_3=4, damaged_ship_3=10, heavy_dmg_ship_3=16
  Man-o-War (yellow): enemy_ship_4=6, damaged_ship_4=12, heavy_dmg_ship_4=18
  Ghost Ship (black): enemy_ship_5=2, damaged_ship_5=8, heavy_dmg_ship_5=14
  Boss ships: boss_ship_std=24, boss_ship_dmg=20, boss_ship_heavy=19

3   | Man-o-War          | 200    | 25      | false  | false     | enemy_ship_4 / damaged_ship_4 / heavy_dmg_ship_4

HP scaling: maxHp = baseHp * (1 + (level-1) * 0.25)
Damage sprite thresholds: >66% normal, 33-66% dmg1, <33% dmg2
Boss spawn level: max(1, round(3 * (1 + (run-1) * 0.3)))

=== BOSS HEALING (enemies.js + kraken.js) ===
Enemy fields: healsUsed, maxHeals, healThresholds[], _justHealed timer
In takeDamage(): if hpRatio <= next threshold, heal 30% maxHp (enemies) or 25% (Kraken)
  Lieutenant: 1 heal at 50%
  Blacktide:  2 heals at 66% and 33%, also laysMines=true
  Kraken:     3 heals at 75%, 50%, 25%

=== MINES (combat.js) ===
Array: this.mines[] = { x, y, damage, life:30, radius:60, pulsePhase }
Laid by Blacktide via enemy.shouldLayMine() (3-7s cooldown)
Player proximity check -> explosion + damage + screen shake
Drawing: 8-spike shape with pulsing red center

=== KRAKEN (kraken.js) ===
Procedural boss - tentacles, body, eyes drawn with canvas
HP: 750 * (1 + (level-1)*0.3), heals 3x
Red eye mode: cycles 8-13s normal -> 4-6s red
  When red: eyes glow red, shots are explosive (blastRadius:80)
  Explosive shots detonate on expiry even if miss, dealing 60% damage in radius
Attacks: tentacle slam (6-10s), ink attack (4-7s)
AI: chase/circle-strafe, never leashes

=== EXTRA CANNON UPGRADE (upgrades.js + player.js) ===
EXTRA_CANNON_COSTS = [25000, 100000]
Available after boss defeats: bossesDefeatedThisRun > extraCannons
player.cannonCount getter: 1 + extraCannons
Multi-cannon: fires cannonCount projectiles per side with 0.08 rad spread
Persistent: saved in player._savePersistent()

=== COMBAT SYSTEM (combat.js) ===
CombatSystem: projectiles[], mines[]
Player auto-fires from port/starboard sides, multi-cannon support
Explosive projectiles: { explosive:true, blastRadius:80 } - detonate on expiry
Kraken fires via _fireKrakenCannon() - explosive when redEyeMode
Enemy kills: gold + cargo drops, boss bonus (25% heal + 2x gold)
Run boss defeat -> game.onRunBossDefeated() -> victory fanfare + finish button

=== STORY SYSTEM (story.js) ===
3 runs with RUN_CHAPTERS{1,2,3}:
  Run 1 "The Corsair's Awakening": chapters at 0/1500/3000/5000m, boss at 8000m (spawnBoss=1)
  Run 2 "The Dark Fleet": chapters at 0/2000/5000m, boss at 9000m (spawnBoss=2)
  Run 3 "The Kraken's Abyss": chapters at 0/3000/6000m, boss at 10000m (spawnBoss=3)
Boss chapter dialogues have 3000ms minDisplayTime + screen shake
unlockCreepy: triggers creepy music track unlock on boss chapters

=== PLAYER (player.js) ===
Base stats: maxHp=100, maxSpeed=150, cannonDamage=15, cannonRange=250, fireRate=1.0
Damage sprites: >75% player_ship, 40-75% player_ship_dmg, <40% player_ship_heavy_dmg
Persistent: persistentGold, totalRuns, bestDistance, completedRuns, currentRun, extraCannons
Per-run: bossesDefeatedThisRun, bossDefeated, buriedGold
Player naming: namePrefix + playerName (set on first play)

=== UPGRADE SYSTEM (upgrades.js) ===
RUN_UPGRADES (lost on death): hull, cannons, speed, fireRate, range, cargo, armor, repairSpeed
PERMANENT_UPGRADES: repairUnlock, permDamage, permHealth, permSpeed, permFireRate,
  permCargo, permArmor, startGold, startHp, luck, permRepairSpeed
Extra cannon: canBuyExtraCannon(), getExtraCannonCost(), buyExtraCannon()

=== UI (ui.js) ===
Panels: title, nameInput, port, trade, upgrade (with extra cannon section),
  dialogue (with minDisplayTime + skip), settings, death, bossVictory, finishRun
HUD: HP bar, gold, buried gold, cargo, kills, distance, repair, medicine, bury button
Debug: GOD (999k hp/gold), B1/B2/KRK (spawn bosses), B? (custom level boss)

=== AUDIO (audio.js) ===
Music: discoverMusic() uses parallel HEAD requests (Promise.all) for themes 1-9 + creepy 1-4
Creepy tracks locked by default, unlocked by story (unlockCreepyTrack())
SFX: playCannon, playHit, playExplosion, playBossExplosion, playCoin, playUpgrade,
  playVictoryFanfare, playPortEnter, playDeath (all procedural via Web Audio API)

=== DEBUG (debug.js) ===
Activated by including "abdebugt3st" in player name
Buttons: GOD, B1, B2, KRK, B? (prompts for level, spawns Ghost Ship at that level)
Uses game._getEnemyImports() to access Enemy/ENEMY_TYPES
