HP scaling formula: maxHp = baseHp * (1 + (level-1) * 0.25)
Example: Man-o-War at level 15 = 200 * (1 + 14*0.25) = 200 * 4.5 = 900 HP

Boss spawn level: level = max(1, round(3 * (1 + (run-1) * 0.3)))
  Run 1: level 3 -> Lieutenant HP = 500 * 1.5 = 750
  Run 2: level 4 -> Blacktide HP = 900 * 1.75 = 1575

Kraken HP: 800 * (1 + (level-1) * 0.3)
  At level 5: 800 * 2.2 = 1760

=== CURRENT COMBAT SYSTEM (combat.js) ===
Player fires cannonballs from ship sides (port/starboard).
- _firePlayerCannon(): finds nearest enemy, fires from closest side
- Projectile speed: 350 (player), 250 (enemy)
- Auto-fire when target in range and cooldown ready
- Currently fires ONE cannonball per side per cooldown cycle
- Projectile object: { x, y, vx, vy, damage, life: 1.5, isPlayer, size: 4-5 }

Enemy kill rewards (combat.js _onEnemyKill):
- Gold particles + text
- Boss bonus: extra gold + "BOSS BONUS" text + 25% maxHp heal
- Run boss defeat -> game.onRunBossDefeated()

Kraken kill rewards (_onKrakenKill):
- 3x death explosions, massive gold (3x goldReward)
- "KRAKEN SLAIN!" text, full heal, screen shake

  - Eye whites: #aaffaa ellipse 10x8
  - Pupils: black, animate with sin/cos
  - Eye glow: #00ff44, shadowBlur 15
  - Mouth: dark ellipse below eyes
  - 5 triangular teeth
- HP: 800 * (1 + (level-1)*0.3)
- Attacks: tentacle slam (6-10s), ink attack (4-7s)
- AI: chase mode, circle-strafe when close
- hitRadius: 70, scale: 2.0, glowColor: '#22ffaa'

=== UPGRADES SYSTEM (upgrades.js) ===
RUN_UPGRADES (lost on death):
  hull, cannons, speed, fireRate, range, cargo, armor, repairSpeed
  Cost formula: baseCost * costMult^level

PERMANENT_UPGRADES (persist with persistentGold):
  repairUnlock, permDamage, permHealth, permSpeed, permFireRate,
  permCargo, permArmor, startGold, startHp, luck, permRepairSpeed

=== STORY SYSTEM (story.js) ===
3 runs with distance-triggered chapters:
  Run 1 "The Corsair's Awakening": boss at 8000m -> spawnBoss=1 (Lieutenant)
  Run 2 "The Dark Fleet":          boss at 8000m -> spawnBoss=2 (Blacktide + 3 escorts)
  Run 3 "The Kraken's Abyss":      boss at 10000m -> spawnBoss=3 (Kraken)

Run completion: player.completedRuns.push(runNum), player.currentRun++
Runs unlock sequentially (must complete run N to unlock run N+1)

=== PLAYER STATS (player.js) ===
Base: maxHp=100, maxSpeed=150, cannonDamage=15, cannonRange=250, fireRate=1.0
Damage states: >75% = player_ship, 40-75% = player_ship_dmg, <40% = player_ship_heavy_dmg
Persistent data: persistentGold, totalRuns, bestDistance, completedRuns, currentRun

=== RUN COMPLETION FLOW (game.js) ===
1. Boss/Kraken killed -> onRunBossDefeated(boss)
2. Victory fanfare + "VICTORY!" overlay
3. After 3s: "Finish Voyage" button appears
4. _finishRun(): marks run complete, advances currentRun, saves, returns to title
