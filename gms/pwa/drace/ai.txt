DRace - Dice Race Battle Game
==============================
Mobile-first portrait PWA board game. Players roll a die and choose which square
to land on (1 to roll value ahead). Squares have positive/negative/treasure effects.
First to finish wins. Single player vs AI opponents.

FILE STRUCTURE
==============
index.html          - Entry point. Screens: title, setup, game, results. Overlay container for panels.
manifest.json       - PWA manifest (portrait, standalone)
sw.js               - Service worker: stale-while-revalidate caching strategy
css/style.css       - Full styles. Dark theme using CSS vars (--accent, --bg-dark, etc.)
icons/              - icon-192.svg, icon-512.svg (app icons)
music/              - Optional: theme1.mp3, theme2.mp3, theme3.mp3 (gracefully skipped if missing)

JS MODULES (loaded in order, each is an IIFE or class):
--------------------------------------------------------

js/storage.js - LocalStorage wrapper
  Storage.get(key, fallback)
  Storage.set(key, value)
  Storage.remove(key)
  Storage.getSettings()          - Returns {soundOn, musicOn}
  Storage.saveSettings(settings)
  Storage.hasSeenGuide()         - Whether user dismissed the welcome guide
  Storage.setSeenGuide()

js/audio.js - Sound effects via Web Audio API + music via HTML5 Audio
  Audio.soundOn                  - getter
  Audio.musicOn                  - getter
  Audio.toggleSound()            - Toggles & persists
  Audio.toggleMusic()            - Toggles, starts/stops playback, persists
  Audio.playMusic()              - Plays music files sequentially, skips missing files
  Audio.stopMusic()
  Audio.sfxRoll()                - Dice roll sound
  Audio.sfxLand()                - Landing on square
  Audio.sfxPositive()            - Positive effect arpeggio
  Audio.sfxNegative()            - Negative effect descending tone
  Audio.sfxTreasure()            - Treasure collect fanfare
  Audio.sfxClick()               - UI button tap
  Audio.sfxWin()                 - Victory fanfare
  Audio.init()                   - Sets up audio context resume on first interaction

js/effects.js - Square effect definitions and application
  Effects.EFFECT_TYPES           - Object of all 24 effect type definitions
                                   Each: {id, icon, name, desc, category, weight}
                                   Categories: neutral, positive, negative, treasure
                                   Weight: relative frequency in board generation pool
  Effects.generateBoard(size)    - Creates array of {index, effect} squares.
                                   Square 0 = Start, last = Finish.
                                   Early squares have fewer negative/powerful effects.
  Effects.applyEffect(player, effect) - Applies effect to player, returns:
                                   {message, toastType, extraAction}
                                   extraAction types: move, roll_again, skip_opponent,
                                   swap, teleport, steal

js/dice.js - Dice rolling and rendering
  Dice.roll()                    - Returns random 1-6
  Dice.renderDiceFace(value)     - Renders SVG dots into #dice-face element
  Dice.animateRoll(finalValue)   - Async: animates rapid face changes then lands on final

js/player.js - Player class + constants
  AI_NAMES                       - Array of 8 AI name strings
  PLAYER_COLORS                  - Array of 4 hex color strings (cyan, orange, purple, green)
  Player(index, name, isHuman)   - Constructor. Properties:
    position, treasure, permBonus, tempBonus, doubleNext, shield,
    skipNextTurn, finished, finishOrder, color
  Player.getEffectiveRoll(rawRoll) - Applies permBonus + tempBonus (doubled if doubleNext).
                                     Resets tempBonus/doubleNext. Min result = 1.
  Player.getScore(totalSquares)  - Finish bonus (100/60/30/10) + treasure
  Player.serialize()
  Player.deserialize(data)       - Static factory

js/board.js - Canvas board renderer with zoom/pan
  Board.init(canvasEl, container) - Sets up canvas, input handlers, zoom controls.
                                    Cleans up previous listeners on re-init.
  Board.setBoard(squares, players) - Configures board data, calculates layout,
                                      fits view, starts render loop.
                                      Serpentine layout: 7 columns, alternating row direction.
  Board.setHighlightSquares(indices, callback) - Highlights squares with pulsing glow,
                                                  clicking a highlighted square calls callback(idx)
  Board.clearHighlight()
  Board.focusOnSquare(sqIndex, immediate) - Zooms camera to center on square
  Board.fitBoard()               - Zooms out to fit entire board in view
  Board.destroy()                - Stops render loop, removes event listeners
  Board.getSquareCenter(index)   - Returns {x, y} pixel position for square

js/ai.js - AI decision making
  AI.chooseSquare(player, choices, squares, allPlayers)
    - 15% random choice for unpredictability
    - Otherwise scores each choice: base score from position distance + effect value
    - Considers shield when evaluating negative squares
    - Swap scored contextually (good if behind, bad if ahead)

js/ui.js - UI panels, toasts, screen management
  UI.showOverlay(htmlContent)    - Shows modal overlay with HTML content
  UI.hideOverlay()               - Hides and clears overlay
  UI.showSettings()              - Settings panel: sound/music toggles + how-to-play button
  UI.showHowToPlay()             - How to play guide panel (closable)
  UI.showWelcomeGuide(onContinue) - First-time welcome: asks if user wants guide.
                                     Calls onContinue after dismissal.
  UI.showEffectPanel(effect, playerName, onDismiss) - Shows effect result to human player
  UI.showToast(message, type)    - type: positive|negative|info|treasure. Auto-removes after 2.8s.
  UI.updatePlayerStats(players, currentIndex, totalSquares) - Bottom bar player chips
  UI.updateHUD(player, turn)     - Top bar: current player + turn counter
  UI.showScreen(screenId)        - Switches between title/setup/game/results screens
  UI.setRollEnabled(enabled)     - Enables/disables roll button
  UI.showChoices(choices, squares, callback) - Shows choice buttons, hides dice area
  UI.hideChoices()               - Hides choice buttons, shows dice area
  UI.setupAIPreview(count)       - Shows AI opponent cards in setup screen

js/game.js - Core game loop
  Game.create(playerName, playerCount, size) - Initializes new game:
    generates board, creates players, inits board renderer, starts music
  Game.doRoll()                  - Rolls dice for current player, shows choices.
                                   For AI: auto-selects after delay. For human: enables UI.
  Game.players                   - getter
  Game.squares                   - getter
  Game.currentPlayerIndex        - getter
  Game.boardSize                 - getter
  Game.gameOver                  - getter

  Internal flow:
    checkCurrentTurn() -> handles skip/finished, triggers AI auto-roll
    doRoll() -> animates dice -> calculates choices -> shows selection UI
    landOnSquare(idx) -> moves player -> applies effect -> shows panel/toast
    handleExtraAction(action) -> processes move/roll_again/skip/swap/teleport/steal
    nextPlayer() -> advances currentPlayerIndex, increments turn on wrap
    endGame() -> calculates scores, shows results screen

js/app.js - Entry point, event binding
  Registers service worker
  Initializes audio context
  Runs title screen particle animation (cleans up on re-entry)
  Binds all navigation buttons (new game, back, start, roll, play again, settings)
  Manages setup screen state (player count, board size selection)
  Shows welcome guide on first game start

GAME MECHANICS
==============
- Board: serpentine path of 30/45/60 squares (configurable)
- Players: 2-4 (1 human + AI)
- Turn: roll 1d6 + bonuses -> choose which square to land on (1 to N ahead)
- Effects applied on landing (shield blocks negative effects)
- Extra actions can chain (move effects, roll again)
- Scoring: finish position bonus (100/60/30/10) + collected treasure points
- Game ends when all but 1 player finishes (or all finish)
- AI uses weighted scoring per square effect with 15% random choice

SCREENS
=======

screen-setup   - Player name, player count (2-4), board size (short/med/long)
screen-game    - HUD (player badge, turn), canvas board, dice + choice UI
screen-results - Rankings with scores, play again / main menu buttons

=======================================

- How to Play: multi-section guide
- Welcome Guide: first-time prompt to read guide
- Effect Panel: shown to human player on landing (icon, name, description)
