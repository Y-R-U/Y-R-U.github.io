The Project

This site is a growing collection of apps, games, and experiments — all built to explore what's possible when a developer teams up with AI coding agents.

Built With AI

Every project here was created through collaboration with Claude AI. Earlier projects (2024–2025) were built with Claude Sonnet 4, while newer work in 2026 uses Claude Sonnet / Opus 4.6.

The developer describes the idea, the AI writes the code, and together they iterate until it works. It's a new way to build software — conversational, fast, and surprisingly powerful.

Why?

To push the boundaries of AI-assisted development. Each project is an experiment: How complex can you go? How fast can you ship? What happens when you let the AI drive?

From simple editors to 3D pirate games, from PWAs to kingdom sims — the answer is: further than you'd think.
